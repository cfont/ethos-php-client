Ellucian Ethos Examples Searching Ethos via PHP

These are example scripts to search Ethos

In an effort to help demonstrate plugging in some application to Ellucian Ethos and use the Proxy API to request information from the authoritative application for the Persons Data Model. In my case this is connecting to Banner through Ethos Integration. These scripts are meant to be slightly more complex and more interesting than the KISS examples I've previously made available.
These examples will search for persons using the "Get Person(s) by" type of API requests...

Search by Credential

Choose either Student ID (which we have historically called the SPRIDEN ID) or choose Logon ID (which we have called the 3rd Party ID and is meant to represent your SSO ID... the account someone may use to log in using AD or LDAP or something like that). Then enter the actual value of that credential so we can search for it using Ethos.

Search by Role

Choose one of the roles to search for people that have that role. Then enter the number of results you want to list from that role keeping in mind the smaller the number the faster the search.
